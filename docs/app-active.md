##激活当前设备
===
###请求方式
---

**POST** `http://open.kaolafm.com/v1/app/active`

###认证参数
---
| 参数名称 | 类型    | 是否必需 |描述
|:------- |-------:|:------:|:----|
| appid   | string |   是   |考拉FM提供的应用id，用于标记一个独立的应用
| sign    | string |   是   |考拉FM提供的签名


###请求参数
---

| 参数名称 | 类型    | 是否必需 |描述
|:------- |-------:|:------:|:----|
| deviceid  | string |   是   |标识唯一设备的id，用于请求openid（可使用手机udid,mac地址,vin等）

###返回参数
---

| 参数名称 | 类型    | 描述 
|:------- |-------:|:------:|
| openid   | string |   我方根据合作方的deviceid生成的设备唯一标识  |
